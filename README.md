# Lung Cancer Risk Analysis

Graduate group project completed for HDS 5230: High Performance Computing and Health Artificial Intelligence at Saint Louis University.

## Project Goal
Analyze lung cancer incidence, mortality, and risk factors across U.S. counties using machine learning and high-performance computing techniques.

## Project Progression
- Project 1: Data collection and project definition
- Project 2: Linear regression and profiling
- Project 3: Random Forest with Dask
- Project 4: Support Vector Machine (SVM)
- Project 5: Genetic Algorithm
- Project 6: Random Forest + Joblib
- Project 7: Artificial Neural Networks (ANN) and Recurrent Neural Networks (RNN)
